# jest-study

## 기본지식 메모

- test에서 given, when, then이란 테스트를 작성하는 방식입니다.
  - given은 테스트의 전제 조건을 정의하고,
  - when은 테스트할 행동을 수행하고,
  - then은 테스트 결과를 검증하는 부분입니다.
